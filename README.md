# In The Making

> **A modern developer portfolio that evolves with its creator.**

## Overview

**In The Making** is a futuristic personal portfolio built with **React**, **TypeScript**, and **Vite**. Rather than serving as a static resume, it showcases projects, technical skills, certifications, and continuous learning through an interactive and immersive user experience.

The project reflects the philosophy that great developers are never finished—they are always **in the making**.

---

## Features

* Modern and responsive UI
* Built with React + TypeScript + Vite
* Modular component architecture
* Projects showcase
* Certifications section
* Skills and technology stack
* Development timeline and roadmap
* Theme support
* Interactive developer-inspired interface

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Pixelraider-sudo/In_the_making.git
```

Navigate into the project:

```bash
cd In_the_making
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Production Build

```bash
npm run build
```
